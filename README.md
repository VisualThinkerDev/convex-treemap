# Convex Treemap Generator

A web-based tool for creating Voronoi-based treemap visualizations from CSV data. Supports Adobe Color (.ase) palette import.

## Features

- Upload CSV data or paste directly
- Import Adobe Color palettes (.ase files)
- Adjustable polygon sides (3-12) and corner rounding
- Customizable labels, colors, and backgrounds
- Export as PNG

## Live Tool

Access the tool at: `https://VisualThinkerDev.github.io/convex-treemap/`

## For Students

1. Prepare your data as CSV with columns: `name`, `value`, and optionally `group`
2. Create a color palette at [color.adobe.com](https://color.adobe.com)
3. Download your palette as .ase file
4. Upload both to the tool
5. Customize and export your visualization

## CSV Format

```
name,value,group
Category A,100,Type1
Category B,50,Type1
Category C,75,Type2
```
